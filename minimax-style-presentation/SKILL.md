# MiniMax 科技简约风格演示生成器

## 功能说明

创建具有 MiniMax 官网风格的科技简约风格演示文稿，支持两种输出格式：

| 格式 | 说明 | 适用场景 |
|------|------|----------|
| **PPT/PPTX** | PowerPoint 演示文稿 | 商务汇报、会议演示 |
| **HTML** | 网页版幻灯片 | 在线分享、网页展示 |

## 设计风格

### 视觉特点

- 科技简约风格，现代 AI 公司通用设计语言
- 深色主调配浅色内容区
- 专业高端感，内容导向

### 配色方案

| 用途 | 颜色 | 说明 |
|------|------|------|
| 主背景 | `#0a0a0a` 或 `#1a1a2e` | 深黑/深蓝黑 |
| 次背景 | `#16213e` | 深蓝 |
| 强调色 | `#e94560` 或 `#00d4ff` | 品牌红/科技蓝 |
| 文字色 | `#ffffff` 或 `#e0e0e0` | 白色/浅灰 |
| 辅助色 | `#533483` | 紫色渐变 |

### 字体

- 中文：思源黑体 / PingFang SC
- 英文：SF Pro / Helvetica Neue / Roboto

### 设计元素

- 简洁的几何图形
- 渐变背景效果
- 模糊玻璃态效果（glassmorphism）
- 简洁的图标风格

## 执行流程

### PPT 生成流程

```
1. 收集需求
   ↓
2. 内容规划（幻灯片结构）
   ↓
3. 调用 pptx-generator Skill 生成PPTX
   ↓
4. 输出文件
```

### HTML 演示生成流程

```
1. 收集需求
   ↓
2. 内容规划
   ↓
3. 调用 PPT Sub-Agent 生成HTML/CSS/JS
   ↓
4. 部署发布
   ↓
5. 提供访问链接
```

## 使用示例

### 示例 1：生成PPT

**用户输入：**
> 帮我做一个关于MiniMax AI技术介绍的PPT，包含公司介绍、技术优势、产品展示三部分

**执行步骤：**
1. 确认输出格式为PPTX
2. 规划幻灯片结构：封面页 → 目录页 → 公司介绍 → 技术优势 → 产品展示 → 总结页 → 结束页
3. 调用pptx-generator skill生成文件
4. 输出 .pptx 文件

### 示例 2：生成HTML演示

**用户输入：**
> 我想做一套网页版的产品演示，主题是2024年度总结

**执行步骤：**
1. 确认输出格式为HTML
2. 规划幻灯片结构和内容
3. 调用PPT Sub-Agent生成HTML/CSS/JS
4. 部署并提供访问链接

## 输出结构

### PPT 结构示例

```
├── 封面页
├── 目录页
├── 第一部分（标题页 + 内容页）
├── 第二部分（标题页 + 内容页）
├── 总结页
└── 结束页
```

### HTML 结构示例

```
slides/
├── index.html          # 主入口
├── slide-01.html       # 封面
├── slide-02.html       # 目录
├── slide-03.html       # 内容页
└── imgs/              # 图片资源
```

## 技术栈

| 格式 | 技术 |
|------|------|
| PPT | PptxGenJS |
| HTML | HTML5 + CSS3 + JavaScript + 响应式设计 |

## 注意事项

1. 始终先确认用户的输出格式需求
2. 幻灯片数量适中，建议6-12页
3. 保持内容简洁，突出重点
4. 中文内容优先使用深色主题
